    Aliro Ed is a software for users who are not familiar with machine learning. Users can experience machine learning tasks on Raspberry Pi 4 with Aliro Ed. Users can download the raspberry pi image from the Aliro Ed website. Users can easily install and run Aliro Ed on Raspberry Pi 4 using this image. Aliro Ed and the Raspberry Pi image provided in this website, are developed by the AI Research Center at the Computational Biomedicine Department Cedars-Sinai Medical Center, Los Angeles, CA, USA.
