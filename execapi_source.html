post /executions

Create a new execution

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • src_code: required(string)
  • dataset_file_id: (string)
  • dataset_id: (string)
  • experiment_id: (string)

Example:

{
  "src_code": "df_new = df.head(10)\ndf_new.to_csv('example.csv')",
  "dataset_id": "444555666",
  "experiment_id": "777888999"
}

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • _id: required(string)
  • src_code: required(string)
  • status: required(one of submitted, completed, error)
  • result: required(string)
  • files: required(array of object)

    Items: file

    • _id: required(string)
    • filename: required(string)
    • mimetype: required(string)
    • timestamp: required(string)
  • _dataset_file_id: (string)
  • _dataset_id: (string)
  • _experiment_id: (string)

{

  "_dataset_file_id": "111222333",
  "_dataset_id": "444555666",

/executions/install post

post

The 'install' command will install a list of python packages via pip. The 'freeze' command will retrive the list of currently installed packages.

post /executions/install

The 'install' command will install a list of python packages via pip. The 'freeze' command will retrive the list of currently installed packages.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • command: required(one of install, freeze)
  • packages: required(array of string)

Examples:

{
  "command": "install",
  "packages": [
    "seaborn"
  ]
}
{
  "command": "freeze"
}

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • command: required(one of install, freeze)
  • packages: required(array of string)
  • exec_results: required(object)
    • stdout: required(string)
    • stderr: required(string)
    • code: required(number)

Examples:

{
  "command": "install",
  "packages": [
    "seaborn"
  ],
  "exec_results": {
    "stdout": "Successfully installed seaborn-0.12.2\n",
